
# 第一个APP

我们开始一个项目时，不一定都要从零开始自己制作，而是可以选择接近的模板，在其上进行修改。

开始项目时，我们提供了九个基础项目，用于开始大家的创作。

### 1.点击主页上方“开始”

也可点击“示例”去选择更多项目

### 2.选择你需要用于开始的项目
